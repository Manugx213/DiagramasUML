# Diagramas UML

Bienvenido al repositorio "DiagramasUML". Este repositorio contiene los ejercicios y ejemplos de diagramas UML realizados durante el curso/práctica.

## ¿Dónde están los ejercicios?

Los ejercicios se encuentran en la carpeta "ejercicios" del repositorio. Puedes acceder directamente aquí:

- Ejercicios (carpeta): https://github.com/Manugx213/DiagramasUML/tree/main/ejercicios

Si prefieres, usa la navegación del repositorio para ver cada ejercicio por separado.

## Estructura sugerida

- ejercicios/       -> Carpeta con los ejercicios numerados (ej: 01, 02, ...)
  - 01/            -> Ejercicio 1 (README, diagramas: .png/.svg, archivos de texto)
  - 02/            -> Ejercicio 2

## Cómo ver los diagramas

1. Abre la carpeta `ejercicios`.
2. Entra en el ejercicio que quieres ver (por ejemplo, `01`).
3. Abre los archivos `README.md` o las imágenes (`.png`, `.svg`) con el visor de GitHub.

## Contribuciones

Si quieres añadir o mejorar ejercicios:

1. Crea una rama nueva: `git checkout -b feat/nuevo-ejercicio`.
2. Añade tu ejercicio bajo `ejercicios/` con una carpeta numerada y un `README.md` explicativo.
3. Abre un Pull Request describiendo los cambios.

Para dudas o comentarios, abre un issue en el repositorio.
